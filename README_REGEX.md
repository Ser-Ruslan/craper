# Регулярные выражения для анализа датасета книг

## Обзор проекта

Этот проект содержит 8 уникальных регулярных выражений, разработанных специально для анализа датасета книг, собранного с сайта books.toscrape.com с помощью веб-скрапера.

## Файлы проекта

- `regex_analysis.py` - Основной скрипт для анализа датасета
- `books_scraper.extracted_books.json` - Датасет с 996 книгами
- `regex_report.md` - Подробный отчет о результатах анализа
- `regex_cheatsheet.md` - Шпаргалка по регулярным выражениям
- `regex_analysis_results.txt` - Результаты анализа в текстовом формате

4. Сохранит результаты в `regex_analysis_results.txt`

## Регулярные выражения

### 1. Поиск цен
```regex
Â£\d+\.\d{2}
```
**Назначение:** Поиск цен в фунтах стерлингов
**Результат:** 996 найденных цен

### 2. Поиск URL
```regex
http://books\.toscrape\.com/catalogue/[^/]+_\d+/
```
**Назначение:** Поиск URL адресов книг
**Результат:** 996 найденных URL

### 3. Поиск рейтингов
```regex
\b(One|Two|Three|Four|Five)\b
```
**Назначение:** Поиск текстовых рейтингов
**Результат:** 1256 найденных рейтингов

### 4. Поиск статусов доступности
```regex
In stock \(\d+ available\)|Out of stock
```
**Назначение:** Поиск информации о наличии книг
**Результат:** 996 найденных статусов

### 5. Поиск издательств
```regex
"([^"]+)"\s*--
```
**Назначение:** Поиск названий издательств в описаниях
**Результат:** 21 найденное издательство

### 6. Поиск категорий/жанров
```regex
\b(?:Fiction|Mystery|Romance|Science|History|Biography|Fantasy|Thriller|Poetry|Art|Music|Travel|Cooking|Business|Self-help|Children|Young Adult|Non-fiction)\b
```
**Назначение:** Поиск популярных категорий и жанров книг
**Результат:** 125 найденных категорий

### 7. Поиск страниц
```regex
\b(?:page|pages|p\.|pp\.)\s+(\d+)\b
```
**Назначение:** Поиск упоминаний страниц в описаниях
**Результат:** 1 найденное упоминание

### 8. Поиск годов
```regex
\b(19|20)\d{2}\b
```
**Назначение:** Поиск годов в описаниях книг
**Результат:** 281 найденный год

### 9. Поиск авторов
```regex
\bby\s+([A-Z][a-z]+(?:\s+[A-Z][a-z]+)*)
```
**Назначение:** Поиск имен авторов после слова "by"
**Результат:** 130 найденных имен
