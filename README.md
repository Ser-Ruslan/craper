# Books to Scrape — учебный скрапер

Скачивает 1050+ страниц (1000 детальных страниц книг + страницы каталога).

## Как запустить
1. `pip install -r requirements.txt` 
2. `python scraper.py`          # собирает ссылки + скачивает (можно прерывать и продолжать)
3. `python parser.py`           # извлекает данные → data/extracted/books.json
4. Открыть `dataset_card.md` 

Данные используются только в учебных целях.
